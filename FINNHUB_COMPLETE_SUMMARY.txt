STEP 5: START FRONTEND (in another terminal)
  cd frontend
  npm run dev

STEP 6: OPEN BROWSER

================================================================================
                         TROUBLESHOOTING
================================================================================

"No FINNHUB_API_KEY set. Using mock data"
  → This is OK! App still works
  → If you want real data: add key to backend/.env
  → Restart backend
  → Done!

"Finnhub API error"
  → App automatically uses mock data
  → Check: internet connection, API key validity, rate limit
  → Wait 60 seconds if rate limited
  → App keeps working!

"Backend won't start"
  → Check Python 3.8+ installed
  → Run: pip3 install -r requirements.txt
  → Try: python3 -m uvicorn main:app --port 8001

"Frontend won't start"
  → Check Node.js 16+ installed
  → Run: npm install
  → Try: npm run dev -- --port 3000

"No real data loading"
  → Verify API key in backend/.env is correct
  → Check backend logs for "✓ Fetched... from Finnhub"
  → Restart backend after editing .env
  → Check internet connection
